How to Login

Username 1 Unit: Tenant's First Initial + Last Name

Note: Accounts are assigned 1 unit to 1 account. If you have multiple units, each account login is the Tenant's First Initial + Last Name + Unit Number

Password: Tenant's Gate Access Code + Last 4 Digits of Phone Number

Example for 1 Unit:

John Smith's Gate Codeode is 1234 and Phone Number is 555-555-5555

John's Username: jsmith

John's Password: 12345555

Example for Multiple Units:

John Smith's Gate Code is 1234, Phone Number is 555-555-5555, and the Unit Number is A27

John's Username: jsmitha27

John's Password: 12345555
